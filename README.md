Опис проекту
Цей додаток — це менеджер користувацьких налаштувань, створений для демонстрації роботи з локальним сховищем SharedPreferences. Він дозволяє керувати профілем, виглядом інтерфейсу (тема, шрифт, мова) та сповіщеннями. Усі дані автоматично зберігаються на пристрої та залишаються доступними після перезапуску додатку.

Реалізовані функції

    Singleton Service: Створено клас PreferencesManager для керування даними в одному екземплярі.

    CRUD операції: Реалізовано збереження, читання, оновлення та видалення налаштувань.

    Автозбереження (Auto-save): Дані зберігаються миттєво при будь-якій зміні в UI.

    Real-time оновлення: Зміна теми (світла/темна) або розміру шрифту одразу застосовується до всього додатку.

    Експорт: Можливість переглянути збережені дані у форматі JSON.

    Скидання: Кнопка "Скинути до типових" очищає всі дані з пам'яті.

Технічні деталі

    Локальне сховище: Пакет shared_preferences: ^2.2.2.

    Структура: Код розділено на UI та сервіс роботи з даними (Singleton Pattern).
